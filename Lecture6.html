First Principle: The Web is More Than Text and Images

Websites need more than static text and images. HTML5 introduced <video> and <audio> tags to embed media without plugins like Flash or Silverlight.

Key Benefits:

  • Standardized and plugin-free playback
  • Cross-browser compatibility
  • Better security and performance

2Ô∏è‚É£ Video Tag

Embedding Video

The simplest video embed:

<video src="my-video.mp4" controls></video>

Attributes:

  • controls: Display play/pause, volume, and fullscreen controls
  • width & height: Set player dimensions
  • autoplay: Starts video automatically (use muted to avoid browser blocking)
  • loop: Repeats video automatically
  • poster: Image shown before video plays

3Ô∏è‚É£ Audio Tag

Embedding Audio

The <audio> tag works like <video> but without a visual component.

<p>Listen to our theme song:</p>
<audio src="audio/theme-song.mp3" controls></audio>

Supports same attributes as video: controls, autoplay, loop, muted.

5Ô∏è‚É£ Two Ways to Embed Video

Source Method vs Src Method

Example 1: Source Method (Professional)

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

Example 2: Src Method (Simple)

<video src="a.mp4" controls></video>

Comparison:

Feature Source Method Src Method
Browser Compatibility Excellent (multiple formats) Good, single format only
Flexibility High Low
Simplicity Complex Very simple
Fallback Custom text possible None
Use Case Public-facing websites Quick tests or internal projects
