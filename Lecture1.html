1Ô∏è‚É£ How the Internet Works

Overview
The Internet is a network of networks, connecting billions of devices worldwide.
  • Direct Link: Early computers connected with simple cables.
  • Switch: Enabled Local Area Networks (LAN).
  • Router: Connected multiple networks across the globe.
  • IP Address: Every device needs a unique identifier.
  • TCP/IP: Protocol suite that ensures data reaches safely in packets.

Fun fact: The first message ever sent over ARPANET in 1969 was "LO" (they intended to type LOGIN but the system crashed). üöÄ

5Ô∏è‚É£ DNS (Domain Name System)

How DNS Works
  1. Browser checks cache (local or router).
  2. If not found, query goes to Recursive Resolver (ISP).
  3. Resolver asks Root ‚Üí TLD ‚Üí Authoritative Server.
  4. Server responds with IP, cached for reuse.

Fun fact: Without DNS, we would type 142.250.184.142 instead of google.com üò≤
