1️⃣ Nested Lists

Overview & Notes

HTML supports two types of lists:

  • Unordered List (<ul>): Items marked with bullets (•)
  • Ordered List (<ol>): Items marked with numbers/letters (1, 2, 3...)
✅ We can nest lists inside list items to represent hierarchy.

Unordered Nested List Example

<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
  <li>Vegetables
    <ul>
      <li>Carrot</li>
      <li>Spinach</li>
    </ul>
  </li>
</ul>

Ordered Nested List Example

<ol>
  <li>Step 1
    <ol>
      <li>Substep 1a</li>
      <li>Substep 1b</li>
    </ol>
  </li>
  <li>Step 2
    <ol>
      <li>Substep 2a</li>
      <li>Substep 2b</li>
    </ol>
  </li>
</ol>

2️⃣ HTML Tables

Overview & Notes

A table is used to display data in rows & columns.

  • <table> → wrapper element
  • <tr> → table row
  • <th> → table header (bold, centered)
  • <td> → table data cell

Simple Table Example

<table border="1" cellpadding="5" cellspacing="0">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Emma</td>
    <td>30</td>
    <td>London</td>
  </tr>
</table>

3️⃣ Advanced Tables (Thead, Tbody, Tfoot)

Example 
<table border="1" cellpadding="5" cellspacing="0">
  <thead>
    <tr>
      <th>Product</th>
      <th>Price</th>
      <th>Stock</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Laptop</td>
      <td>$1000</td>
      <td>50</td>
    </tr>
    <tr>
      <td>Mouse</td>
      <td>$25</td>
      <td>200</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="2">Total Products</td>
      <td>250</td>
    </tr>
  </tfoot>
</table>

4️⃣ Merge Cells (Colspan & Rowspan)

Example 
<table border="1" cellpadding="5" cellspacing="0">
  <tr>
    <th rowspan="2">Name</th>
    <th colspan="2">Marks</th>
  </tr>
  <tr>
    <th>Math</th>
    <th>Science</th>
  </tr>
  <tr>
    <td>John</td>
    <td>95</td>
    <td>88</td>
  </tr>
</table>
