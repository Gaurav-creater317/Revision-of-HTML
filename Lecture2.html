Definition & Overview
HTML (HyperText Markup Language) is the backbone of web development. - HyperText: Connects documents via links (like Google search results). - Markup: Uses tags like <p>, <h1>, <img> to give structure & meaning.

Fact: HTML was invented by Tim Berners-Lee in 1991. The first website is still live today!
