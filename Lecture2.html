1️⃣ What is HTML?

Definition & Overview
HTML (HyperText Markup Language) is the backbone of web development. - HyperText: Connects documents via links (like Google search results). - Markup: Uses tags like <p>, <h1>, <img> to give structure & meaning.

Fact: HTML was invented by Tim Berners-Lee in 1991. The first website is still live today!

8️⃣ Video Example 🎬

Watch Video

Attributes: - controls: Adds play/pause UI
- autoplay: Auto-plays on load
- loop: Repeats automatically
- muted: Starts muted (important for autoplay)
- fullscreen: Users can toggle full-screen mode (usually built-in with controls)

🔹 Embedding External Videos

You can embed YouTube or Vimeo videos inside your webpage using the <iframe> tag.

💡 Tip: Use allowfullscreen in <iframe> to let users watch in full screen.
