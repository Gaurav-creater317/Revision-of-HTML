Relative vs Absolute Paths (Most Common Interview Question)
File paths tell the browser where your files (HTML, CSS, images, JS) are stored. Relative paths are based on project location. Absolute paths use full web addresses.

ğŸ”¹ Relative Paths

  • Same folder: <a href="about.html">About</a>
  • Inside sub-folder: <img src="images/logo.png" alt="Logo">
  • From parent folder: <img src="../images/logo.png">

ğŸ”¹ Absolute Paths

  • Full URL: <a href="https://www.google.com">Google</a>
  • External Image: <img src="https://upload.wikimedia.org/.../html5.svg">
